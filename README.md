SimpleCode
==========

Utils to help create easily maintainable code

Using with Maven
----------------


```
<repositories>
    <repository>
        <id>simple-code-mvn-repo</id>
        <url>https://raw.github.com/cbensemann/SimpleCode/mvn-repo/</url>
        <snapshots>
            <enabled>true</enabled>
            <updatePolicy>always</updatePolicy>
        </snapshots>
    </repository>
</repositories>
```
